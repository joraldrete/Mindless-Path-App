# Mindful Path: Health and Mental Wellness Tracker

Mindful Path is a Python-based wellness tracking application designed to help individuals build healthier habits, improve their daily routines, and reflect on their overall well-being. Developed as a Computer Science Capstone Project
---

## Features

- Daily Habit Tracking:  
  Log essential activities such as:
  - Exercise (minutes)
  - Sleep (hours)
  - Water intake (glasses)
  - Calories consumed

- User-Friendly Design:  
  Built with the `wxPython` GUI framework for a simple, clean, and intuitive interface.
